# 3. Bảng Điều Khiển (Dashboard)

## Tổng Quan

Sau khi đăng nhập, bạn được đưa đến **Bảng điều khiển** — trung tâm quản lý tất cả lộ trình học của bạn.

Giao diện bảng điều khiển bao gồm:

```
┌─────────────────────────────────────────────────────────┐
│  [Logo MeMap]           [Tìm kiếm]        [Avatar]      │
├───────────┬─────────────────────────────────────────────┤
│  Sidebar  │                                             │
│           │   Lộ trình của tôi          [Tạo mới]      │
│  - Trang  │   ┌──────┐  ┌──────┐  ┌──────┐            │
│    chủ    │   │ Card │  │ Card │  │ Card │            │
│           │   └──────┘  └──────┘  └──────┘            │
│  - Học    │                                             │
│    tập    │   Được chia sẻ với tôi                     │
│           │   ┌──────┐  ┌──────┐                       │
│  - Tài    │   │ Card │  │ Card │                       │
│    khoản  │   └──────┘  └──────┘                       │
└───────────┴─────────────────────────────────────────────┘
```

---

## Phần "Lộ Trình Của Tôi"

Hiển thị tất cả lộ trình mà **bạn tạo ra** hoặc **có quyền sở hữu**.

### Thẻ lộ trình (Card)

Mỗi lộ trình hiển thị dưới dạng thẻ với các thông tin:

- **Ảnh bìa** — Hình thu nhỏ của lộ trình
- **Tên lộ trình**
- **Tên chủ sở hữu**
- **Số lượt yêu thích** (⭐)
- **Ngày tạo**

### Tác vụ trên thẻ

Nhấn vào thẻ lộ trình để **Xem lộ trình**. Ngoài ra, bạn có thể nhấn vào menu (⋮) trên thẻ để:

| Tác vụ | Mô tả |
|--------|-------|
| **Xem** | Mở lộ trình ở chế độ xem |
| **Yêu thích / Bỏ yêu thích** | Đánh dấu lộ trình |
| **Chia sẻ** | Mời người khác hoặc thay đổi quyền truy cập |
| **Xoá** | Xoá lộ trình (chỉ chủ sở hữu, không thể hoàn tác) |

---

## Phần "Được Chia Sẻ Với Tôi"

Hiển thị tất cả lộ trình mà **người khác đã chia sẻ** với bạn — ví dụ, giáo viên chia sẻ lộ trình học với học sinh.

- Bạn có thể **xem** và **học** những lộ trình này
- Bạn **không thể chỉnh sửa** lộ trình của người khác
- Nếu bị xoá quyền truy cập, lộ trình sẽ biến mất khỏi danh sách này

---

## Tìm Kiếm Lộ Trình

Sử dụng thanh tìm kiếm ở đầu trang để tìm lộ trình theo:
- **Tên lộ trình**
- **Mô tả**

Kết quả được lọc ngay khi bạn gõ.

---

## Phân Trang

Mỗi trang hiển thị tối đa **6 lộ trình**. Dùng nút điều hướng ở cuối danh sách để xem thêm.

---

## Điều Hướng Chính (Sidebar / Navigation)

| Mục | Trang |
|-----|-------|
| **Trang chủ / Dashboard** | Bảng điều khiển |
| **Học tập** | Danh sách bài kiểm tra có thể làm |
| **Tài khoản** | Hồ sơ, gói cước, lưu trữ |

---

## Thông Báo

Biểu tượng chuông (🔔) trên thanh điều hướng hiển thị thông báo mới:
- Lộ trình được chia sẻ với bạn
- Kết quả bài kiểm tra
- Cập nhật từ giáo viên
